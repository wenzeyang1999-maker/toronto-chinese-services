## 1. 平台是什么

NCC 北美华人圈是一个**面向多伦多华人的一站式生活服务平台**，让留学生 / 打工人 / 新移民可以：

- **找到本地华人服务**：搬家、保洁、报税、接送、装修、家教等
- **接单赚外快**：用自己的技能/车/时间在平台上帮其他华人，换钱
- **看招聘、租房、二手、活动**：所有生活需求一站搞定
- **进社区互助**：在多村论坛里发问、晒经验、求推荐

平台同时具备 **微信群的人情味** 和 **app 的效率**。

## 2. 新用户三分钟快速上手

### Step 1：打开网站

电脑或手机浏览器访问 **https://toronto-chinese-services.vercel.app**

### Step 2：注册账号

1. 右上角点 **注册**
2. 填邮箱、设置密码、填名字（手机号可填可不填）
3. 朋友的邀请码？有就填，没有就跳过
4. 点 **注册** → 系统会自动登录你

**预期结果**：自动跳到首页，右上角变成"我的"图标

### Step 3：开启位置权限（重要）

首次访问首页或地图时，浏览器会问"是否允许获取位置"，**点允许** → 才能：
- 看到"附近"的服务
- 在地图上看到自己的位置（红色圆点）
- 按距离排序

### Step 4：浏览首页

首页四大入口：

| 入口 | 用途 |
|------|------|
| **🏠 生活服务** | 搬家、保洁、接送、装修、汽车维修等（点开有子分类弹窗）|
| **♻️ 二手交易** | 家具、汽车、装备等转让 |
| **🎪 大多广场** | 同城活动、集市、公益 |
| **💬 多村论坛** | 社区聊天、求推荐、经验分享 |

### Step 5：试试找服务

1. 首页中部"找服务/找客户"切换栏，确保是 **找服务**
2. 下面会显示**附近热门服务**列表（如果开了定位会按距离排序）
3. 上方有 **列表/地图** 切换 — 点 **地图** 看分布
4. 看到中意的，点卡片进详情页

### Step 6：联系服务商

在服务详情页：

- **看资料**：照片、价格、评价、联系方式（电话 + 微信）
- **微信沟通**：点绿色微信按钮，自动复制服务商微信号到剪贴板
- **站内消息**：点"发消息"按钮 → 进入聊天页（实时对话）
- **电话联系**：点电话图标自动拨号

**结果**：服务商收到你的消息会通过邮件 + 浏览器推送通知，及时响应。

## 3. 核心概念：用户模式 vs 服务商模式

NCC 是双向平台 — 每个人都可以**既是用户又是服务商**。在"我的"页面顶部切换。

### 视觉区分

| 模式 | 背景色 | 用户身份 |
|------|--------|---------|
| **用户模式** 🟢 | 淡绿色背景 | 我要找服务、找帮手 |
| **服务商模式** 🔵 | 淡蓝色背景 | 我要接单、卖技能 |

### 切换方法

进入"我的"页面 → 头像下方有一张**切换卡片**：

```
┌─────────────────────────────────────────────┐
│  当前模式                                    │
│  👤 用户模式            [切换为服务商模式]   │
└─────────────────────────────────────────────┘
```

点右侧按钮一次，整页背景从绿色平滑变蓝色（300ms 过渡），左侧标签变成"服务商模式"，按钮文案变成"切换为用户模式"。

### 两种模式的菜单差异

**用户模式（淡绿）显示这些菜单：**
- 帐号和安全
- 我的收藏 ❤️
- 我的关注 👥
- 我的报价请求 📋
- 我的消息 💬
- 浏览记录 🕐
- 会员等级 👑
- 邀请好友 🎁
- 通知设置 🔔

**服务商模式（淡蓝）显示这些菜单：**
- 我的主页 📊（封面 · 简介 · 标签）
- 联系方式与资质验证 ✓
- 我的消息 💬
- 会员等级 👑
- 邀请好友 🎁
- 通知设置 🔔

## 5. 用户模式（找服务）完整流程

### 5.1 我有明确的服务需求

例：家里马桶堵了，需要找师傅

**方法 A：搜索关键词**
1. 首页顶部搜索框输入"管道疏通"或"水管"
2. 进入搜索结果页 — 看到匹配的服务列表
3. 点开有兴趣的服务进详情页

**方法 B：地图找最近的**
1. 首页 → 切到 **地图** 视图
2. 看绿色 🟢 pin（在线接单师傅）— 这些是当前可以立刻响应的
3. 点 pin 弹窗 → 看姓名、技能、点击进主页

**方法 C：浏览类别**
1. 首页 **生活服务** 入口点开
2. 弹窗里选具体子类（如"水电维修"）
3. 自动跳到对应类别页

### 5.2 我不知道找谁，让师傅来找我

这是"发布需求"的杀手锏 — 你只发一条需求，所有匹配的服务商会主动来联系你。

1. 点首页悬浮按钮 **+ 发布需求**
2. 填写：
   - **标题**：例"急需搬家帮手，周末可用"
   - **类别**：搬家
   - **详细描述**：补充时间、要求、数量
   - **区域**：选你所在地区
   - **预算**：例 `$80-120` 或 `面议`
   - **有效期**：7 / 14 / 30 天
3. 点 **发布需求**
4. 内容审核（约 1 秒）→ 成功

**结果**：
- ✅ 你的需求出现在所有服务商的"发现客户"地图 + 列表
- ✅ 符合标签的服务商会**立即收到推送通知**（声音 + 浏览器通知）
- ✅ 师傅看到后通过"去聊聊"主动给你发消息
- ✅ 你可以在"我的 → 我的报价请求"看到匹配的师傅列表

### 5.3 查看 / 收藏 / 关注

**收藏服务**：详情页右上角心形 ♡ → 变红 ❤️ → 进"我的 → 我的收藏"找

**关注服务商**：进服务商主页 → 点 **关注** 按钮 → 服务商发新内容时你会收到邮件通知

**浏览历史**：自动记录最近看过的 20 条 → "我的 → 浏览记录"

### 5.4 评价与举报

服务结束后：

1. 进服务详情页 → **评价** 区域
2. 给 1-5 星 + 写文字评价
3. 提交 → 服务商收到邮件通知

**举报**：
- 服务页右上角小旗子 🚩 → 选举报原因（虚假/欺诈/不当内容等）→ 提交
- 聊天页面也有 🚩 举报按钮 + 🚫 拉黑按钮

## 6. 服务商模式（接单赚钱）完整流程

### 6.1 切换到服务商模式

"我的" → 点 **切换为服务商模式** → 背景变蓝 → 看到服务商专属菜单

### 6.2 第一步：完善我的主页

点 **我的主页**：

| 内容 | 怎么填 | 建议 |
|------|--------|------|
| **封面图** | 点上传 | 平台自动裁成 3:1，传清晰的工作场景照 |
| **头像** | 账号设置里改 | 头像 = 信任 |
| **简介 Bio** | 一句话介绍 | 例："7 年搬家经验，国语粤语都行，自带货车" |
| **业务标签 skill_tags** | 输入回车添加 | 关键！例：`搬家` `接送` `装修` |
| **社交链接** | 微信、IG、小红书 | 添加 = 客户信任感 ↑ |

**特别提醒：业务标签很重要！** 因为：
- 系统按这些标签**自动匹配**符合的客户需求
- 客户发布需求时，标签命中你的关键词，**你的手机会立刻响**（声音 + 推送）
- 也会出现在"发现客户"地图上

### 6.3 第二步：通过资质验证

点 **联系方式与资质验证**：

- **邮箱验证**：注册时已完成
- **手机验证**：填手机号 + 验证码（暂时只支持北美号码）
- **商户认证**（可选）：上传营业执照 → 管理员 24h 内审核 → 通过后服务卡片上显示"商户认证"徽章

**结果**：认证越多 → 客户信任越高 → 接单更多

### 6.4 第三步：发布我的服务

首页 → 悬浮 **+ 发布服务** 按钮 → 进 PostService 表单：

1. **服务标题**：例"专业搬家 - 国语师傅，自带货车"
2. **类别**：搬家
3. **价格**：`$50/小时` 或 `$200 起` 或 `面议`
4. **服务区域**：多选（北约克、万锦、Downtown 等）
5. **详细描述**：服务内容、流程、注意事项
6. **上传图片**：最多 6 张（建议工作照、车辆照）
7. **位置**（选填）：精确位置 → 地图上能 pin 出来
8. **发布**

**结果**：
- ✅ 内容 AI 审核（1-2 秒）
- ✅ 上架成功页 → 可点 **分享给朋友** 复制链接发微信群
- ✅ 你的服务出现在首页、类别页、地图、搜索结果
- ✅ 所有"关注你"的用户**收到邮件通知**

### 6.5 第四步：发现客户（核心）

"我的 → 切换到服务商模式" → 回到首页 → 顶部 tab 变成：

```
[找服务]  [发现客户  🔴3]
```

红色数字 3 = **自上次查看以来新增的、且匹配你标签的需求数**。

点 **发现客户** tab：

- **列表视图**：所有需求按时间排，红色边框=紧急（≤3天）
- **地图视图**：橙色 pin 散布在你 3-5-10 km 范围内（可调滑杆）
  - **只显示匹配你 skill_tags 的需求**
  - 你的红色圆点 = 你的位置
  - 点 pin 弹窗看详情 + 跳转

### 6.6 第五步：接到推送 → 联系客户

**实时提醒系统**：

| 你的状态 | 触发条件 | 你会收到 |
|---------|---------|---------|
| **app 开着** | 任何用户发布匹配标签的新需求 | 🔊 双音 ding-ding + 浏览器通知 + 站内 toast |
| **app 关着** | 同上 | 📱 手机系统推送通知（需要先在通知设置里开权限）|

收到通知 → 点开通知或直接进 app → 看到需求详情 → 点 **去聊聊** → 进入聊天页 → 跟客户沟通

### 6.7 数据看板

进 **我的主页** 滚下来看：

- **帮过 X 人**
- **平均回复时间 X 小时**（自动统计，影响"快速回复"徽章）
- **收到 X 个评价 / 平均 4.X 星**

## 7. 地图功能详解

地图是平台最重要的差异化功能。两种打开方式：

### 7.1 内嵌地图（首页）

首页 → "找服务" 或 "发现客户" → 点 **地图** 切换

### 7.2 全屏地图

任意地图视图 → 右上角 **⛶ 全屏** 图标 → 进入 `/map` 页面

全屏地图：
- 顶部搜索框（实时过滤所有 pin）
- 左上角显示数量
- 右下角定位按钮
- 地图占满全屏（无干扰）

### 7.3 Pin 颜色含义

| 颜色 | 含义 | 谁看得到 |
|------|------|---------|
| 🔴 红圈（带白色边） | 你自己的位置 | 所有人 |
| 🔵 蓝色 pin | 普通服务 | 用户模式 |
| 🔴 红色 pin | 推广服务 | 用户模式 |
| 🟢 绿色 pin | 在线接单的师傅 | 用户模式 |
| 🟠 橙色 pin | 客户发的需求（求服务）| 服务商模式 |

### 7.4 距离滑杆（服务商模式专属）

在"发现客户"地图视图，上方有：

```
距离范围  [3 km]  [5 km]  [10 km]    ⓘ 已按你的标签过滤：搬家、接送
```

点 3/5/10 切换搜索半径，选择持久化（下次进来自动是你上次的选择）。

## 8. 消息与通知

### 8.1 站内消息

**进入方式**：
- 顶部"消息"按钮（桌面）
- 底部 nav 中的 💬 消息（移动）
- "我的 → 我的消息"

**功能**：
- 实时对话（基于 Supabase Realtime，无延迟）
- 发送失败自动显示"重试"按钮 ↺
- 对方未读时你看到红点
- 右上角可举报 🚩 或拉黑 🚫 对方
- 可直接复制对方微信号 / 拨打电话

### 8.2 邮件通知

平台会自动发邮件给你的：

| 事件 | 收件人 |
|------|-------|
| 收到新消息 | 接收方 |
| 有人关注了你 | 服务商 |
| 收到新评价 | 服务商 |
| 服务下有人提问 | 服务商 |
| 你关注的人发了新内容 | 关注者 |
| 匹配你需求的师傅响应了 | 客户 |

### 8.3 推送通知（Web Push）

**开启方法**：
1. "我的 → 通知设置"
2. 看到提示"开启浏览器通知" → 点击 → 浏览器弹权限 → 允许
3. 下面 4 个开关变成蓝色（默认全开）：
   - 新消息通知
   - 报价匹配通知
   - 评价通知
   - 平台公告

**触发条件**：
- 任何会发邮件的事件 + 你的浏览器/手机授权了通知 → 收推送
- **特别强调**：匹配你 skill_tags 的新需求 → 即使 app 关着也会推送

### 8.4 关闭通知

回到 "通知设置" → 关掉任何一个开关 → 该类事件不再推送（邮件仍会发）

## 9. 社区圈子

类似小红书 + 即刻的本地社区。

### 9.1 浏览

`/community` → 看所有人发的帖（搬家经验、租房推荐、生活问题、转让等）

3. 上传图片（最多 4 张）
4. 选类别 + 区域
5. 发布 → AI 审核 → 上线

- ❤️ 点赞（每人每帖只能 1 次）
- 💬 评论（也支持回复评论）
- 🔄 分享（复制带文案的链接发微信）
- ✏️ 编辑（自己发的帖可以改）
- 🚩 举报不当内容

### 9.4 关注后新内容推送

关注一个用户 → 他/她发新社区帖 → 你收到邮件 + 推送

## 11. 移动端 PWA 安装

NCC 是 PWA（Progressive Web App），可以"装"到手机/电脑桌面像 app 一样用。

### 11.1 自动提示安装

首次访问会底部弹一个 banner：

```
📥 把 NCC 装到主屏幕
   收消息更及时，打开更快
                              [安装]  [✕]
```

点 **安装** → 浏览器原生安装对话框 → 确认 → 桌面/主屏幕出现 NCC 图标

### 11.2 手动安装

如果错过了 banner，进 "我的" 页面底部，永久有一张：

```
📥 安装到桌面
   添加为 App，打开更快、收消息更及时
```

点击：

- **iPhone Safari**：弹"添加到主屏幕"3 步图文引导
- **安卓 Chrome / Edge / 电脑 Chrome**：直接弹原生安装对话框
- **其他浏览器**：弹通用引导（地址栏 ⊕ 图标 / 浏览器菜单"添加到主屏幕"）

### 11.3 安装后的好处

- ✅ 桌面/主屏幕直接点图标启动（不用打开浏览器找网址）
- ✅ 全屏运行，没有浏览器的地址栏占空间
- ✅ Web Push 推送通知更可靠（iOS 必须装了才能收推送）
- ✅ 部分页面离线可访问（Service Worker 缓存）

## 12. 安全与隐私

### 12.1 内容审核

所有发布的内容（服务、需求、帖子）都经过 **AI 审核**：

- 政治内容、违法内容、欺诈、仇恨言论 → **自动拒绝**
- 模糊内容 → 倾向放行（避免误伤）
- 审核失败时用户会看到具体原因

### 12.2 举报与拉黑

**举报**：见到不当内容点旗子 🚩 → 选原因 → 24h 内管理员审核

**拉黑**：聊天页右上角 🚫 → 确认 → 对方再也无法给你发消息（DB trigger 强制拒绝）

### 12.3 隐私

- 你的邮箱、手机号默认**不公开**
- 服务详情页只显示你选择公开的联系方式（微信号、电话）
- 头像和名字是默认公开的
- 删除账号 → 所有数据级联删除

### 12.4 平台安全

- 所有密码 bcrypt 加密 + Supabase Auth 托管
- 数据库 Row Level Security 全面启用
- HTTPS 全站
- 所有 admin 高危操作走 RPC + 审计日志
- 推送服务用 VAPID 签名防伪造

## 14. 常见问题 FAQ

### Q1：注册时收不到验证邮件？
A：检查垃圾邮件文件夹。如果还是没有，可能 Resend 服务在你的邮箱被拒，换 Gmail/Outlook 试试。

### Q2：地图上看不到我的位置？
A：浏览器没授权位置权限。Chrome 地址栏左边小图标 → 网站设置 → 位置 → 允许。

### Q3：推送通知收不到？
A：3 步检查：
1. "通知设置"开关都打开了吗？
2. 浏览器通知权限给了吗？
3. iOS：必须先把 NCC 装到主屏幕（PWA）才能收推送

### Q4：我发布的服务/帖子被审核拒绝了？
A：AI 检测到敏感词。原因会显示在错误信息里。修改后重新提交。

### Q5：服务商模式下我看不到任何客户需求？
A：检查"我的主页 → 业务标签 skill_tags" — 没设标签会过滤掉所有需求。设上"搬家""接送"等关键词后回到首页"发现客户"地图就有了。

### Q6：怎么知道我是不是登录状态？
A：右上角如果显示"我的"图标就是登录了；显示"注册/登录"按钮就是没登录。

### Q7：忘记密码怎么办？
A：登录页点"忘记密码" → 输入注册邮箱 → 收邮件 → 点链接 → 设新密码。

### Q8：能用手机号登录吗？
A：UI 有手机号登录入口，但后端的 Phone Auth 尚未配置（需要短信服务商）。目前请用邮箱登录。

### Q9：邀请好友能赚什么？
A：你的邀请码被人注册时填写 → 你的"已邀请人数"+1 → 达到阈值会自动升级会员等级（trigger 已配置）。

### Q10：拉黑了一个人，对方知道吗？
A：对方不会收到通知，但他给你发消息会被静默拒绝（DB 层 trigger）。
