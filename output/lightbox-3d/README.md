# OPIUM Lightbox 3D Pack

This folder contains a lightweight 3D mockup scene for upload into viewers such as Sketchfab.

Files:
- `opium-lightbox-scene.obj`: scene geometry
- `opium-lightbox-scene.mtl`: material bindings
- `assets/opium-front.png`: front lightbox artwork
- `assets/opium-side.png`: side panel artwork
- `assets/toronto-dt-bg.png`: Toronto downtown nighttime background

Recommended upload:
1. Zip this whole folder.
2. Upload the zip to Sketchfab or another OBJ-compatible viewer.
3. In the viewer, tweak:
   - camera angle
   - environment light
   - material glow / emissive strength
   - sign rotation / scale if needed

Notes:
- This is a presentation mockup scene, not a fabrication-accurate engineering model.
- The lightbox body is intentionally simple so it is easy to edit downstream.
